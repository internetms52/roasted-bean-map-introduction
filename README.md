# 咖啡熟豆地圖

收錄台灣各烘豆坊熟豆資訊，支援 AI 助理透過 MCP 直接查詢豆單。

🌐 **網站：** https://roasted-bean-map.6826646.xyz

---

## 功能

- 依烘焙程度、處理法、風味標籤篩選咖啡豆
- 收錄粒豆咖啡、柏克斯咖啡、哈巴夏咖啡、氧氣咖啡、再再之在等店家豆單
- 提供 MCP 介面，讓 AI 助理可以直接查詢

---

## 在 Claude 使用 MCP 查詢

### 連線設定

1. 進入 Claude.ai **Settings → Integrations**
2. 新增 MCP Server，輸入以下網址：

```
https://roasted-bean-map.6826646.xyz/mcp
```

3. 連線後即可直接用自然語言詢問 Claude

### 範例提問

- 「幫我找衣索比亞的淺焙咖啡豆」
- 「有哪些日曬處理的豆子？」
- 「推薦有花香風味的咖啡豆」
- 「找找看有厭氧處理、帶莓果風味的豆子」

### 支援的查詢條件

| 條件 | 可用值 |
|------|--------|
| 烘焙程度 | 淺焙、中淺焙、中焙、中深焙、深焙 |
| 處理法 | 日曬、水洗、蜜處理、厭氧等 |
| 風味標籤 | 花香、莓果、柑橘、巧克力等 |
| 豆單名稱 | 關鍵字模糊搜尋 |
